## 📝 الكود الذي تم تعديله

### من هنا:
```typescript
// كود قديم - نفس التعليمات لكل منتج
const systemPrompt = `أنت مساعد صيدلي...
📋 تعليمات التعامل مع الأسئلة:
1️⃣ للسؤال الأول...
`;
```

### إلى هنا:
```typescript
// كود جديد - تعليمات مخصصة
const productNames = context.match(/📦 المنتج: ([^\n]+)/g);

const dataInstructions = `
⚡ البيانات المتاحة: ${productNames.join(', ')}
👉 لا تخلط البيانات بين المنتجات!
`;

const systemPrompt = `...${dataInstructions}...`;
```
