## 🔎 كيفية معرفة ما إذا كان المنتج موجود؟

### الطريقة 1: اختبر مباشرة
- اسأل في التطبيق
- إذا قال "غير موجود" ← غير موجود
- إذا أعطاك معلومات ← موجود

### الطريقة 2: تحقق من Supabase
1. افتح [https://app.supabase.com](https://app.supabase.com)
2. انقر SQL Editor
3. اكتب:
```sql
SELECT * FROM clinical_embeddings 
WHERE content ILIKE '%[المنتج]%' 
LIMIT 5;
```

### الطريقة 3: اسأل في التطبيق بشكل واضح
```
"هل لديكم [اسم المنتج]؟"
```
