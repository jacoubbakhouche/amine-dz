# 🚀 دليل الاختبار العملي المباشر

## 🎯 الهدف
اختبار تطبيق الاستشارات الصيدلانية للتأكد من أنه:
1. **يجيب من البيانات فقط** (بدون اختراع معلومات)
2. **يفهم السياق الكامل** (المحادثات السابقة)
3. **يعالج التحذيرات بحذر** (يضعها أولاً)
4. **لا يضع معلومات مختلقة**

---

## 📋 الخطوات التحضيرية

### 1. تشغيل التطبيق
```bash
cd /Users/bakhouche/amin
npm install  # إذا لم تثبت المكتبات
npm run dev
```

### 2. فتح المتصفح
```
http://localhost:5173/chat
```

### 3. التحقق من حالة التحميل
- [ ] الصفحة تحملت بنجاح
- [ ] أيقونة "Get Started" مرئية
- [ ] حقل الكتابة موجود في الأسفل

## 🧪 الاختبارات التفصيلية

## ✅ **الاختبار 1: منتج معروف موجود**

### خطوات الاختبار:

1. اكتب في حقل الرسالة:
```
معلومات عن ELUGEL GEL BUCCAL
```

2. اضغط **Send** أو انقر الزر

3. انتظر الإجابة (قد تستغرق 5-10 ثواني في المرة الأولى)

### ما يجب أن تراه:
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📦 المنتج: ELUGEL GEL BUCCAL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

معلومات عن:
- المادة الفعالة: Chlorhexidine
- الاستخدام: علاج الالتهابات
- الجرعة: 2-3 مرات يومياً
- الحد الأقصى: 15 يوم

⚠️ تحذيرات:
- استشر الطبيب قبل الاستخدام المطول
```

### ✅ علامات النجاح:
- [ ] الإجابة حقيقية من البيانات
- [ ] المنتج اسمه صحيح (ELUGEL)
- [ ] المادة الفعالة مذكورة
- [ ] التحذيرات موجودة
- [ ] الجرعة واضحة
- [ ] لا معلومات مختلقة

### ❌ علامات الفشل:
- ❌ إجابة عامة غير محددة
- ❌ معلومات لا علاقة لها
- ❌ استشهاد بمنتجات غير موجودة
- ❌ بدون تحذيرات

**النتيجة:** ✅ / ❌

## ✅ **الاختبار 2: منتج غير موجود**

### خطوات الاختبار:

1. اكتب:
```
هل لديكم دواء Doliprane؟
```

2. اضغط Send

3. انتظر الإجابة

### ما يجب أن تراه:
```
عذراً، منتج Doliprane غير مدرج في قاعدة بياناتنا السريرية.

المنتجات المتاحة لديي:
- ELUGEL (علاج الالتهابات)
- ELVEOR (تبييض الأسنان)
- ELMEX (منتجات الفلورايد)

يرجى مراجعة الصيدلي أو طبيب الأسنان.
```

### ✅ علامات النجاح:
- [ ] يقول "غير موجود" بوضوح
- [ ] لا يختلق معلومات عن Doliprane
- [ ] يعرض بدائل من البيانات
- [ ] يقترح استشارة الطبيب
- [ ] رد احترافي وحازم

### ❌ علامات الفشل:
- ❌ يحاول الإجابة عن Doliprane (اختراع)
- ❌ يقول "قد تكون هناك" (غير مؤكد)
- ❌ يقترح منتجات غير موجودة
- ❌ رد غير واضح

**النتيجة:** ✅ / ❌

## ✅ **الاختبار 3: سؤال متابع (السياق)**

### خطوات الاختبار:

**الخطوة 1:** اكتب الرسالة الأولى:
```
أخبرني عن ELUGEL
```

**الخطوة 2:** انتظر الإجابة

**الخطوة 3:** اكتب السؤال المتابع:
```
كم مرة أستخدمه يومياً؟
```

**الخطوة 4:** انتظر الإجابة

### ما يجب أن تراه:
```
حسب معلومات ELUGEL:

💊 عدد مرات الاستخدام: 2-3 مرات يومياً

⏱️ المدة القصوى: 15 يوم

📌 ملاحظة مهمة:
لا تتجاوز المدة المحددة دون استشارة طبيب
```

### ✅ علامات النجاح:
- [ ] فهم أن السؤال عن ELUGEL
- [ ] الإجابة من البيانات الصحيحة
- [ ] لم يكرر كل المعلومات السابقة
- [ ] تركيز على السؤال المحدد
- [ ] منطقي ومرتبط بالسياق

### ❌ علامات الفشل:
- ❌ يسأل "أي منتج تقصد؟"
- ❌ يعيد الإجابة الكاملة
- ❌ معلومات غير متعلقة
- ❌ عدم فهم السياق

**النتيجة:** ✅ / ❌

## ✅ **الاختبار 4: طلب بدائل**

### خطوات الاختبار:

**الخطوة 1:** اكتب:
```
أريد معلومات عن ELVEOR
```

**الخطوة 2:** انتظر الإجابة

**الخطوة 3:** اكتب:
```
هل هناك منتجات أخرى للتبييض؟
```

### ما يجب أن تراه:
```
نعم، لدينا منتجات تبييض أخرى:

━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. ELVEOR BLANCHEUR
━━━━━━━━━━━━━━━━━━━━━━━━━━━
المزايا: تبييض فعال
الجرعة: كما موضح

━━━━━━━━━━━━━━━━━━━━━━━━━━━
2. ELMEX OPTIWHITE
━━━━━━━━━━━━━━━━━━━━━━━━━━━
المزايا: تبييض تدريجي
الجرعة: كما موضح

📊 الفرق:
- كلاهما فعال للتبييض
- الاختلاف في المادة الفعالة
```

### ✅ علامات النجاح:
- [ ] يذكر منتجات حقيقية من نفس الفئة
- [ ] معلومات من البيانات
- [ ] مقارنة واضحة
- [ ] لا منتجات مختلقة

### ❌ علامات الفشل:
- ❌ يقترح منتجات غير موجودة
- ❌ معلومات غير دقيقة
- ❌ يقول "لا أملك معلومات"
- ❌ بدون مقارنة

**النتيجة:** ✅ / ❌

## ✅ **الاختبار 7: جرعة آمنة للأطفال**

### خطوات الاختبار:

اكتب:
```
ما هي الجرعة الآمنة لطفل عمره 4 سنوات؟
```

### ما يجب أن تراه:
```
لحساب الجرعة الآمنة، أحتاج إلى معرفة:

1️⃣ اسم المنتج أو الدواء
   - مثال: ELUGEL، Amoxicillin

2️⃣ وزن الطفل (اختياري)
   - يساعد في حساب دقيق

📚 الجرعات الآمنة للأطفال:
حسب قاعدة البيانات للعمر 4 سنوات:

🎯 [المنتجات المتاحة للعمر 4 سنوات]

⚠️ تحذير مهم:
استشر طبيب الأسنان أو الصيدلي دائماً قبل إعطاء أي دواء للأطفال.
```

### ✅ علامات النجاح:

- [ ] جرعات آمنة من البيانات
- [ ] تحذيرات واضحة

- ❌ معلومات غير آمنة
- ❌ بدون تحذيرات
